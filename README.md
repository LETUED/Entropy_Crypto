# Entropy_Crypto

Multiscale Permutation Entropy(MPE)로 암호화폐 시장의 "질서 있는 구간"을 찾아 진입하는 트레이딩 전략 연구. 2026-05~06, 실험 35개로 검증한 뒤 연구를 종결했다.

## 결론부터

**검정한 모든 메커니즘에서, 거래비용을 넘는 견고한 엔트로피 기반 우위는 없었다.**

초기 walk-forward에서 Sharpe +1.391이 나왔다. 하지만 거래단위 재측정과 블록 부트스트랩을 적용하니 이 수치는 측정 인플레이션이었다 — 희소한 시간당 수익 곡선을 √8760으로 연율화한 산물에, 특정 설정 조합 의존이 겹친 것. 거래단위로 다시 재면 Sharpe 0.6~0.78이고 신뢰구간에 0이 들어온다. 37건의 거래로는 "엣지=0"을 기각할 수 없었다. 실거래 자본을 투입하기 직전에 여기서 멈췄다.

| 메커니즘 | 실험 | 결과 |
|---------|------|------|
| 상승추세 눌림목 매수 (방향 알파) | Exp30~33 | ≈0. CI에 0 포함, 정체는 조건부 베타 |
| 코인·기간 확장으로 유의성 확보 | Exp31C, 32 | 실패. n=373에서도 net 음수 |
| 레짐 게이트로 손실 꼬리 제거 | Exp33 | 실패. 손실이 진입 메커니즘에 내재 |
| MPE → 미래 변동성 예측 | Exp34 | 통과 (유일한 양성, 룩어헤드 검증 완료) |
| 변동성 신호의 사이징 가치 | Exp35 | 경제적 가치 0 |
| 약세장 평균회귀 (직교 메커니즘) | Exp31B | 결정적 음성. CI 전체가 0 아래 |

돌아보면 원전략의 가정이 정반대였다. 저MPE를 "안전한 진입 구간"으로 썼는데, 실제로 저MPE는 미래 고변동의 예고 신호였다(Exp34). 전략이 유일하게 양(+)으로 기울던 구간도 "강세장에서 고베타 코인 롱"과 구분되지 않았다 — 알파가 아니라 베타.

실험 전 과정과 최종 감사는 [docs/RESEARCH_LOG.md](docs/RESEARCH_LOG.md)에 있다.

## 전략 (최종형)

- 대상: BTC, SOL, AVAX, ADA, DOT — 1H 캔들, 롱 전용
- 진입: `MPE < 10%ile` AND `RSI < 30` AND `가격 > MA200` AND `온체인 엔트로피 < 40%ile`
- 청산: `RSI > 50` 또는 168시간 경과
- 사이징: Kelly 단계형 (MPE 1%ile → 50%, 5%ile → 30%, 10%ile → 15%)

고정 파라미터 (전 실험 공통):

```python
m = 3, tau = 1, scales = [1, 2, 4, 8]   # MPE
window = 168                             # 롤링 윈도우 (시간)
MAX_HOLD_H = 168
MA_PERIOD = 200
FEE_RATE = 0.001
```

## 검증 방법

- walk-forward: 학습 12개월 → 테스트 6개월, 6구간 롤링
- 순열검정 10,000회 + 블록 부트스트랩 5,000회 (PnL 신뢰구간)
- 거래단위 지표로 통일 — 시간당 곡선의 연율화 금지 (8장에서 초기 수치 정정)
- 룩어헤드 점검을 별도 실험으로 강제 (Exp34는 이 관문을 통과한 유일한 신호)
- LightGBM + SHAP로 피처 기여 분해 (Exp26b)

## HFT 틱 트랙

저빈도 트랙과 병행으로, Binance aggTrade 스트림의 Shannon 엔트로피 + 오더북 임밸런스(OBI)로 틱 단위 매매를 실험했다(HFT-Exp01~04, 페이퍼 트레이딩 실측 포함).

결과는 수학적으로 닫혔다. 방향을 100% 맞히는 오라클을 가정해도 적자다. 병목이 방향이 아니라 이동 크기이기 때문 — 틱 이동폭 0.02~0.05%가 왕복 수수료 0.04~0.10%보다 작다(MFE p90 = 수수료의 17%). 상세는 [docs/hft_research_log_20260526.md](docs/hft_research_log_20260526.md).

## 실행

