# 1. 연구 배경 및 철학적 출발점

## 1.1 라마누잔으로부터의 영감

이 연구의 철학적 출발점은 인도의 수학자 스리니바사 라마누잔(1887~1920)이다. 라마누잔은 정규 교육 없이 수학의 가장 심오한 공식들을 직관으로 발견했으며, 그 특징은 다음과 같다.

- 증명 없이 패턴을 직관으로 발견
- "왜 맞는지 모르지만 맞음" (블랙박스)
- 나중에 다른 수학자들이 수십 년에 걸쳐 증명

이는 현대 AI/머신러닝이 데이터에서 패턴을 발견하는 방식과 구조적으로 동일하다. 라마누잔이 "신이 알려줬다"고 표현한 직관적 패턴 발견이, 현대에는 AI가 데이터에서 수행하고 있다.

## 1.2 라마누잔 머신 (Ramanujan Machine)

이스라엘 테크니온 공대에서 실제로 구현된 AI 시스템으로, 라마누잔처럼 증명 없이 추측 공식을 생성한다. 가우스가 평생 걸려 발견한 파이 공식을 몇 시간 만에 재발견하고, 수십 개의 새로운 공식도 발견했다.

2025년에는 Euler2AI 프로젝트로 확장되어 수십만 개의 논문을 스캔해 94%의 공식들 사이에서 새로운 연결고리를 자동으로 발견했다.

**연결점:** 라마누잔 머신이 수학 상수에서 패턴을 찾듯이, 이 연구는 시장 엔트로피에서 패턴을 찾는 것을 목표로 한다.

## 1.3 메타물질로부터의 영감

메타물질(Metamaterial)은 자연에 없는 물질 구조를 인공적으로 설계하여 자연 법칙은 그대로 유지하면서도 구조 자체가 새로운 성질을 만들어내는 물질이다 (음굴절, 투명망토 등).

이 연구의 핵심 접근법은 메타물질과 동일하다:
- 데이터 자체는 변하지 않음
- 데이터를 보는 **구조를 바꾸면** 기존에 안 보이던 패턴이 보임

| 메타물질 | 이 연구 |
|---|---|
| 자연 물질 | RSI (기존 지표) |
| 인공 구조 설계 | 텐서 + 엔트로피 프레임 |
| 음굴절 (새로운 성질) | 기존에 안 보이던 패턴 |
| 빛을 다르게 휘어지게 함 | 신호를 다르게 해석함 |

# 2. 핵심 가정 및 직관

## 2.1 두 가지 핵심 가정

**가정 1:** 주식장(및 코인장)은 예측 불가능한 엔트로피 시스템이다.

**가정 2:** 엔트로피는 근사로 예측할 수 있다.

이 두 가정의 결합이 이 연구의 핵심이다. 가정 1이 맞더라도 가정 2가 성립하면, 엔트로피가 낮아지는 구간을 찾아 그 구간에서만 예측을 시도할 수 있다.

## 2.2 주식장의 본질에 대한 직관

주식장에서 돈을 버는 구조는 다음과 같이 이해된다:

- 이기고 지는 확률이 반반인 도박장에서 자금관리를 통해 이득을 보는 구조
- 집단 광기, 우주의 엔트로피, 불확실한 공간 속에서 정확한 예측은 불가능
- 그러나 **근사로 예측**하는 것은 가능

이는 라마누잔이 P(200)을 정확한 증명 없이 근사로 발견한 것과 같은 철학이다.

## 2.3 엔트로피와 예측 가능성의 관계

```
엔트로피 낮음 → 질서 → 예측 가능 → 베팅
엔트로피 높음 → 무질서 → 예측 불가 → 관망
```

실증 연구에 의해 확인된 사실:
- 엔트로피가 낮은 구간에서 ML 알고리즘의 예측 정확도가 높아짐
- 엔트로피 최하위 5% 구간에서 평균의 2.89배 변동폭 발생
- 극단적 이벤트(폭락, 위기) 때 엔트로피가 오히려 감소하는 역설

## 2.4 집단 광기에 대한 관점

집단 광기(Herding Behavior)는:
- 계산의 대상이 아님
- 집단 광기가 만들어내는 **결과물인 엔트로피 패턴**을 측정

```
광기 (원인) → 측정 불가, 신경 안 써도 됨
        ↓
엔트로피 변화 (결과) → 측정 가능
        ↓
RSI (강도 측정) → 기존 도구로 방향 확인
```

# 3. 라마누잔의 유산과 연결

## 3.1 모의 세타 함수 (Mock Theta Functions)

라마누잔이 죽기 직전 마지막 편지에 남긴 17개의 함수로, 80년 동안 수학자들이 이해하지 못하다가 2002년 즈웨거스(Zwegers)에 의해 "조화 마스-형식(Harmonic Maass Form)의 정칙 부분"임이 밝혀졌다.

**핵심 구조:**
```
모의 세타 함수 (불완전)
        +
보정항 (비정칙 부분)
        =
조화 마스-형식 (완전한 구조)
```

**3차 모의 세타 함수 예시:**
$$f(q) = \sum_{n=0}^{\infty} \frac{q^{n^2}}{(-q;q)_n^2}$$

## 3.2 모의 세타 함수와 엔트로피의 연결

직접적 연결은 아니지만, 분할함수 경로를 통해 수학적으로 연결된다:

```
라마누잔 모의 세타 함수
        ↓ (분할함수 경로)
통계역학 엔트로피 구조
        ↓ (응용)
주식장 엔트로피 계산
        ↓
RSI + 텐서로 구현
```

라마누잔의 분할함수, 모듈러 형식, 모의 세타 함수가 블랙홀 열역학과 통계역학에서 이미 활용되고 있으며, 이 체인이 수학적으로 말이 된다는 것이 검증되었다.

## 3.3 모의 세타 함수의 핵심 성질과 시장 대응

**성질 1: 임계점 근방 거동**
- 단위근 근방에서 폭발적으로 변하는 구조
- 시장 대응: RSI 70(과매수)/30(과매도) 임계점 근방의 엔트로피 급변
- 실증: 엔트로피가 낮아지는 구간에서 변동폭 2.89배 증가 (검증됨 ✅)

**성질 2: 불완전하지만 규칙적**
```
모의 세타 함수 (불완전) + 보정항 = 완전한 구조
RSI 단독 (불완전 신호) + ??? 보정항 = 완전한 예측
```
- 구조적으로 말이 됨 ✅

**성질 3: 더 큰 구조의 일부**
- 모의 세타 함수가 80년 뒤 더 큰 구조의 일부였음이 밝혀짐
- RSI도 더 큰 엔트로피 텐서 구조의 일부일 수 있음 ✅

## 3.4 라마누잔 공식의 역할 정리

라마누잔 공식은 직접 적용의 대상이 아니라 **철학적 영감**으로 활용된다:

```
라마누잔 → 철학적 영감
         "불완전한 신호에서 숨겨진 구조 찾기"
              ↓
         실제 도구
         엔트로피 계산
```

# 4. 주식장 엔트로피 이론

## 4.1 주식장을 엔트로피 시스템으로 보면

$$H_{market}(t) = -\sum_{i} p_i(t) \log p_i(t)$$

여기서 $p_i(t)$는 시점 t에서 각 가격 상태의 확률.

## 4.2 브라운 운동 모델과의 관계

주식의 일일 등락은 실제로 동전던지기에 가까우나, 완전한 랜덤은 아니다. 이를 수식으로:

$$dS = \mu S \, dt + \sigma S \, dW$$

- $\mu$ = 드리프트 (시장의 우상향 편향)
- $\sigma$ = 변동성
- $dW$ = 완전한 랜덤 부분 (위너 프로세스)

## 4.3 켈리 기준 (Kelly Criterion)

반반 도박에서도 자금관리만으로 장기 수익이 가능하다는 수학적 증명:

$$f^* = \frac{p(b+1) - 1}{b}$$

- $p$ = 이길 확률
- $b$ = 배당률
- $f^*$ = 최적 베팅 비율

## 4.4 집단 광기의 수학적 모델

투자자들의 군집 행동을 입자 모델로 수학화한 연구에서:
- 기대수익률 = 위상공간의 위치
- 선호도 = 위상공간의 속도

로 해석하여 군집 행동을 수식화. 이를 통해 시장의 엔트로피가 낮아지는 구간(집단이 같은 방향으로 움직이는 구간)을 수학적으로 포착할 수 있다.

# 5. 엔트로피 계산 방식 탐색

## 5.1 후보 방식들

### 5.1.1 섀넌 엔트로피 (Shannon Entropy)
$$H = -\sum_{i} p_i \log p_i$$

- 가장 기본적인 형태
- 극단적 이벤트 때 엔트로피가 감소하는 역설 발견
- 장점: 직관적, 계산 쉬움
- 단점: 노이즈에 약함, 시계열 순서 무시

### 5.1.2 근사 엔트로피 (Approximate Entropy, ApEn)
$$ApEn(m, r, N) = \phi^m(r) - \phi^{m+1}(r)$$

- 시계열의 불규칙성 측정
- 노이즈에 강함, 짧은 데이터에도 작동
- 단점: 파라미터(m, r) 선택이 까다로움

### 5.1.3 샘플 엔트로피 (Sample Entropy, SampEn)
$$SampEn(m, r, N) = -\ln\frac{A}{B}$$

- 근사 엔트로피 개선판
- 자기 자신과의 비교 제거 → 더 정확
- 샘플 엔트로피와 ML 예측 정확도 사이의 상관관계 최초 증명

### 5.1.4 순열 엔트로피 (Permutation Entropy, PE) ⭐ 핵심 후보
$$PE = -\sum_{\pi} p(\pi) \log p(\pi)$$

- 시계열의 순서 패턴을 기호로 변환해서 계산
- 예: [1,3,2] → "상승 후 하락" 패턴
- 1929년, 1987년, 2008년, 2020년 코로나 등 4대 주식 폭락 전 패턴 발견
- **조기경보 지표**로 활용 가능
- 장점: 계산 빠름, 노이즈 강건, 비선형 패턴 포착
- 단점: 진폭 정보 손실

### 5.1.5 멀티스케일 순열 엔트로피 (Multiscale PE) ⭐⭐ 핵심 후보
$$MPE(\tau) = PE(\text{시간 스케일 } \tau \text{에서})$$

- 순열 엔트로피를 여러 시간 스케일에서 동시에 계산
- 단기/중기/장기 패턴 동시 포착
- GARCH 같은 기존 변동성 모델보다 **34% 높은 민감도**
- RSI 다기간(7일, 14일, 21일)과 자연스럽게 연결

### 5.1.6 주문흐름 엔트로피 (Order Flow Entropy)
$$H_{flow} = -\sum_{states} p_{transition} \log p_{transition}$$

- 매수/매도 주문의 전이 패턴을 마르코프 행렬로 표현
- 실시간 계산 가능, 미시구조 포착
- **방향은 예측 못하고 크기만 예측** (치명적 한계)
- 수학적으로 방향 구분 불가 (순열 불변성)

## 5.2 엔트로피 방식 비교표

| 방법 | 계산 난이도 | RSI 연결성 | 방향 예측 | 조기경보 |
|---|---|---|---|---|
| 섀넌 | ⭐ | 중간 | △ | △ |
| 근사(ApEn) | ⭐⭐ | 높음 | ○ | ○ |
| 샘플(SampEn) | ⭐⭐⭐ | 높음 | ○ | ○ |
| 순열(PE) | ⭐⭐ | **매우 높음** | ○ | **◎** |
| 멀티스케일 PE | ⭐⭐⭐ | **매우 높음** | ○ | **◎** |
| 주문흐름 | ⭐⭐⭐⭐ | 중간 | ✗ | ○ |

## 5.3 엔트로피의 치명적 한계

엔트로피는 **방향을 수학적으로 구분할 수 없다.**

순열 불변성(Permutation Invariance): 매수/매도 라벨을 바꿔도 엔트로피 값이 동일하기 때문에, 방향 예측 정확도가 45%로 동전던지기 수준이다.

수익의 87.8%가 "언제 변동이 크냐"를 아는 것에서 나오고, 방향 예측 기여는 0.0%이다.

**따라서:**
```
엔트로피 → 언제 예측 가능한지 (타이밍 필터)
RSI → 방향 확인 (방향 신호)
```
이 두 요소의 역할 분리가 핵심이다.

# 6. 기존 퀀트 트레이딩과의 차별점

## 6.1 기존 퀀트 트레이딩

르네상스 테크놀로지스 등 헤지펀드들이 사용하는 방식:

- RSI를 단순 팩터로 입력값으로 사용
- RSI + 선형 모델 조합
- RSI + 딥러닝 모델

**문제점:** 고정된 공식 → 시장이 학습하고 무력화

## 6.2 이 연구의 접근법

| | 기존 퀀트 | 이 연구 |
|---|---|---|
| **핵심 질문** | "패턴이 뭔가?" | "언제 패턴이 생기는가?" |
| **RSI 사용** | 신호 그대로 사용 | 엔트로피 낮을 때만 신호 신뢰 |
| **시장 적응** | 고정 공식 → 무력화 | 엔트로피 자체가 적응형 |
| **노이즈 처리** | 필터링 어려움 | 엔트로피 높으면 자동 관망 |
| **수학 기반** | 통계/확률론 | 정보이론 + 물리학 |
| **예측 대상** | 가격 방향 직접 예측 | 예측 가능한 순간을 먼저 찾음 |
| **공식 적응** | 시장이 학습하면 무력화 | 엔트로피 법칙은 물리 법칙 → 무력화 불가 |

**핵심 차이 한 줄 요약:**
> 기존 퀀트: **"무엇을 예측할까?"**
> 이 연구: **"언제 예측이 가능한가를 먼저 판단한다"**

## 6.3 불확정 공식의 개념

기존 퀀트의 문제: 고정된 공식 → 시장이 적응하면 무력화

해결책: **엔트로피 자체가 공식이 되면?**

```
시장이 어떻게 변하든
엔트로피는 항상 계산 가능
        ↓
엔트로피가 낮아지는 순간
= 패턴이 생기는 순간
= 항상 유효 (물리 법칙이므로)
```

시장이 어떤 공식에 적응해도 엔트로피 법칙은 피할 수 없다.

# 7. 앙상블 및 동적 통합 시스템

## 7.1 단순 앙상블 (기본)

여러 엔트로피 계산 방식을 동시에 사용:

```
섀넌 엔트로피       →  H1
순열 엔트로피       →  H2
멀티스케일 엔트로피 →  H3
샘플 엔트로피       →  H4
        ↓
H_total = w1*H1 + w2*H2 + w3*H3 + w4*H4
```

## 7.2 앙상블보다 좋은 방식들

### 7.2.1 EntroPE (Entropy-Guided Dynamic Patch Encoder)
- 엔트로피로 구간을 동적으로 분할
- 엔트로피 낮은 구간 → 촘촘하게 분석
- 엔트로피 높은 구간 → 듬성듬성 분석
- ETTh1에서 20%, Electricity에서 15% 정확도 향상
- **한계:** 주식 방향 예측에 직접 적용된 연구 없음
- **역할:** 예측 가능 구간에서만 선택적으로 적용

### 7.2.2 엔트로피 기반 베이지안 모델 평균 (En-BMA)
- 엔트로피 자체로 가중치를 결정하는 구조
- 높은 엔트로피 = 낮은 가중치 (자동 조정)
- 사전 가정 없이 작동
- **한계:** 주식 직접 적용 연구는 수문학(강수량) 쪽에만 있음

### 7.2.3 RL 동적 가중치
- 강화학습으로 가중치를 실시간 조정
- 과거 예측 성능에 따라 보상/패널티
- 시장 변화에 자동 적응
- **한계:** 과적합 위험 존재

## 7.3 통합 시스템 구조

```
주식/코인 데이터 입력
        ↓
[멀티스케일 순열 엔트로피]
단기/중기/장기 동시 계산
        ↓
[En-BMA로 통합]
H_total 계산
        ↓
H_total 낮음?
YES → [EntroPE] 촘촘히 분석
    → RSI로 방향 확인
    → [RL] 실시간 적응
    → 매매
NO  → 관망
```

# 9. 온체인 데이터 엔트로피 변환

## 9.1 온체인 데이터의 종류

```
1. 거래소 유입/유출 (Exchange Flow)
2. 고래 트랜잭션 (Whale Transactions)  
3. MVRV (시장가치/실현가치)
4. NVT (네트워크가치/거래량)
5. USDT 플로우 (구매력 지표)
6. 장기 보유자 공급량 (LTH Supply)
```

## 9.2 각 데이터의 엔트로피 변환

### 9.2.1 거래소 유입/유출 엔트로피 (H_flow)

```python
p_inflow  = inflow / (inflow + outflow)
p_outflow = outflow / (inflow + outflow)

H_flow = -(p_inflow * log(p_inflow) 
         + p_outflow * log(p_outflow))
```

- H_flow 낮음 → 유입 or 유출 한쪽으로 쏠림 → 방향성 강함
- H_flow 높음 → 유입/유출 균형 → 혼란 상태

### 9.2.2 고래 트랜잭션 엔트로피 (H_whale)

```python
# 트랜잭션 크기를 구간으로 나눠서
bins = [0, 10, 100, 1000, 10000, inf]  # BTC
p_i = count_in_bin_i / total_transactions

H_whale = -Σ p_i * log(p_i)
```

- H_whale 낮음 → 특정 크기의 거래 집중 → 고래 움직임 감지
- H_whale 높음 → 거래 크기 분산 → 일반 소매 거래

### 9.2.3 MVRV 엔트로피 (H_mvrv)

```python
# MVRV를 시계열로 보고 순열 엔트로피 적용
H_mvrv = PermutationEntropy(MVRV_series, m=3, tau=1)
```

- MVRV가 규칙적으로 움직이면 → H 낮음 → 사이클 패턴
- MVRV가 불규칙하면 → H 높음 → 방향 없음

### 9.2.4 USDT 플로우 엔트로피 (H_usdt)

```python
H_usdt = PermutationEntropy(USDT_netflow_series)
```

- USDT가 거래소로 유입 → 매수 유동성 증가 → BTC/ETH 수익률 양으로 예측
- H_usdt 낮음 → USDT 유입 패턴 규칙적 → 매수 압력 예측 가능

## 9.3 온체인 통합 엔트로피

```python
H_onchain = w1*H_flow + w2*H_whale + w3*H_mvrv + w4*H_usdt
# 각 가중치는 En-BMA로 동적 결정
```

## 9.4 온체인 데이터 직접 접근의 중요성

거래소 API는 거래소가 블록체인 데이터를 이미 가공해서 제공하는 것이다. 진정한 온체인 데이터는 블록체인 노드에 직접 접근해서 가져온다.

coinrandom 프로젝트에서 이미 구현된 접근법:
- 블록 해시 직접 접근 (ETH, BTC)
- 여러 거래소 공개 API 분산 수집 (API 키 불필요)
- 역마르코위츠 최적화로 최소 상관 코인 선택

# 10. 뉴스 감성 엔트로피

## 10.1 뉴스 감성 → 엔트로피 변환

```
뉴스 텍스트
    ↓
감성 분석 (긍정/부정/중립 확률)
p = [p_긍정, p_부정, p_중립]
    ↓
섀넌 엔트로피 계산
H_news = -Σ p_i log p_i
    ↓
H_news 높음 = 시장 의견이 분산됨 (혼란)
H_news 낮음 = 시장 의견이 일치함 (방향성)
```

## 10.2 예시

```
뉴스가 전부 부정적 → p = [0.05, 0.9, 0.05]
H_news 낮음 → 집단 광기 발생 중 → 신뢰도 높음

뉴스가 섞여있음 → p = [0.33, 0.33, 0.33]
H_news 높음 → 혼란 상태 → 관망
```

## 10.3 종목별 반응 시간 문제

주식마다 뉴스에 반응하는 속도가 다르다. FinBERT + 유전 알고리즘으로 종목별 감성 반응 윈도우를 추정하면 승률이 5% 향상되었다는 연구가 있다.

이를 고정값으로 처리하면 오히려 노이즈가 될 수 있으므로, 종목/코인별 반응 시간을 동적으로 추정해야 한다.

이 연구와 방향이 정반대이면서 같은 재료를 사용한다:

|---|---|---|

| 엔트로피 활용 | 예측 불가 활용 | 예측 가능 구간 찾기 |
| 데이터 소스 | 동일 | 동일 |

# 16. 결론 및 핵심 주장

## 16.1 이 연구의 핵심 주장

1. **주식장과 코인장은 엔트로피 시스템이다.** 완전한 예측은 불가능하지만 근사 예측은 가능하다.

2. **엔트로피가 낮아지는 구간이 예측 가능한 순간이다.** 이 구간만 골라서 거래하는 것이 핵심이다.

3. **RSI는 방향을, 엔트로피는 타이밍을 담당한다.** 역할 분리가 이 시스템의 설계 원칙이다.

4. **메타물질 접근법:** 같은 데이터를 다른 구조로 보면 새로운 패턴이 발현된다.

5. **코인장에 더 유리하다:** 온체인 데이터라는 추가 엔트로피 소스와 고래 효과가 시스템을 강화한다.

6. **coinrandom이 인프라다:** 이미 구현된 데이터 수집 파이프라인을 방향만 바꿔서 재활용할 수 있다.

## 16.2 라마누잔으로부터의 교훈

라마누잔이 증명 없이 "맞으면 됨"이었듯이, 이 연구도 수학적 증명보다 **근사로 예측하는 것**을 목표로 한다.

엔트로피 법칙은 물리 법칙이기 때문에 시장이 아무리 변해도 무력화할 수 없다. 이것이 기존 퀀트 공식들과 근본적으로 다른 점이다.
